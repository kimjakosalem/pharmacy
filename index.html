Deploy to Netlify
  1. Go to netlify.com and create a free account
  2. Click Add new site → Deploy manually
  3. Drag and drop the index.html file into the upload box
  4. Go to Site settings → Change site name
  5. Set it to drkimpharmacy to get drkimpharmacy.netlify.app
  6. Share the URL with your cashiers!
