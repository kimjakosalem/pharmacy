Import / Export
CSV columns: sku, name, category, address, suppliers, cost, price, inventory, lowStock, variants, mrpCap, mrpRef
Variants format: size:packaging:cost:price:inventory|size:packaging:cost:price:inventory  ·  Suppliers format: Supplier A,Supplier B
mrpCap / mrpRef: optional DOH price-cap tag (e.g. mrpCap=44.20, mrpRef="DOH MRP-MWP List 22-Jun-2026") — shown as a badge in the table and in Edit Product, and warns before saving a price above it. Leave both columns out entirely to avoid touching existing tags on re-import.
Re-importing: matches rows to existing products by SKU (or by exact name if SKU is blank) and updates them in place — export, edit in a spreadsheet (e.g. variant costs/stock), then re-import to bulk-update instead of creating duplicates. Unmatched rows create new products. Leave a variant's inventory blank to keep it unmigrated (shared stock).

Take-Home cash from a closed shift is automatically put into an envelope here (labeled by date + shift).
